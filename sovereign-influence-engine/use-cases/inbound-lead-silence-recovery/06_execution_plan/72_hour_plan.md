# 72-Hour Implementation Plan

## Day 1
- map existing services and adapters
- define DB deltas and schemas
- create lead-state derivation logic
- create deterministic decision evaluator
- wire dry-run output and logs

## Day 2
- implement action execution adapters
- implement CRM logging
- add scheduler/job runner
- add decision persistence
- add feature flags and per-tenant safeguards

## Day 3
- enable controlled execution on limited sample
- capture outcomes
- validate metrics
- run first calibration cycle
- document next delta and blockers

## Success criteria
- at least one approved action executed end-to-end
- outcomes captured automatically
- decisions auditable
- rollback path documented
